RM.VG URL to iCalendar

This tool generates an iCal file when GET parameters are passed.

The form below can be used to generate the URL.

The following parameters are required to cause a calendar file to be generated: DTSTART, DTEND, summary.

The following parameters are supported:

  • DTSTARTTimestamp of date start.
  • DTENDTimestamp of date end.
  • SUMMARYThe summary/title for the event.
  • LOCATION (optional) A location for the event.

Limitations:

  • Time zone selection is not yet supported, always is America/New_York
  • Only very limited fields are supported at this time.
  • All Day is not supported.
  • GUI: When a timestamp is loaded, it should populate the drop down menus.
  • GUI: There should be a method to make the drop downs ready from a timestamp.
  • GUI: There should be a button to clear the form.
  • GUI: There should be a button to copy the start date to the end date.
